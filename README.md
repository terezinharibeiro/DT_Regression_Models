# R codes and dataset for the paper "A novel approach to generate distributions with applications to regression modeling"
This repository contains R codes and dataset used in the application of the paper "A novel approach to generate distributions with applications to regression modeling" by Dutta, Vila and Ribeiro (2026).

The following R scripts are provided to replicate the results presented in the application section of the paper.

- Application_Foliage.R: R script to replicate the inference and diagnostics results for the Foliage dataset. 
- DTED_GAMLSS.R:  R function that fits RDTED regression models using the gamlss package.
- DTWD_GAMLSS.R:  R function that fits RDTWD regression models using the gamlss package.
